**场景**：产品经理在做季度规划，想快速了解几款任务管理工具的功能和定价差异，下周要在会议上讲，不需要完整报告。


> **AI**：好的，走快查模式，不创建文件，直接给你结论。

> 我理解你的目标是：**评估三款工具对小团队（5-20人）的适用性，重点看功能覆盖和定价**。

>
> | 功能 | Notion | Linear | Jira |
> |------|--------|--------|------|

> | 子任务 | ✓ | ✓ | ✓ |
> | 时间线/甘特 | ✓（付费）| ✓ | ✓ |

> - Notion Plus：$10/人 → $50/月，免费版限协作
> - Linear Business：$8/人 → $40/月，免费版功能完整
> - Jira Standard：$7.75/人 → $38.75/月，免费版限10人

> 1. **Linear 免费层最慷慨**：免费版无人数限制且功能完整，小团队冷启动成本最低

>
> **推荐**：小团队从零开始 → Linear（免费起步，体验最流畅）；团队已重度使用文档协作 → Notion；工程团队且重视 Git 工作流集成 → Linear 或 Jira
